
=== Template organisation ===

/template
    /action
        /admin 
            Templates for administration/privieged actions.   
        /core  
            Templates for actions that work with the core database
        /hplc  
            HPLC action templates
        /ms    
            Mass spec action templates
        /nmr   
            NMR action templates
    /common
        Various common & default presentation templates for the basic 
        user interface.
    /lib
        Template code & macro libraries written in Freemarker.
    /ui
        /experiment
            Templates for the basic "experiment perspective" UI, most of
            which are based on templates in /common. These are mainly 
            templates for the header, navigation & context panel.
        /query
            Templates for the basic "query perspective" UI.
        /user
            Templates for the basic "user summary perspective".
