Common 1D/2D plotting options

Output type

  • HPGL  - This is basic HPGL code, which can be read by many drawing programs, or  HPGL code embedded in PCL5  control code (when the Include PCL5 control codes option has been checked in the Options box). This file can be submitted directly to a PCL printer.
  • BW Postscript - Black and White postscript. This option is usually preferred for black and white output, because colors are often translated into dotted or dashed lines.
  • Color Postscript  - Full color Postscript output.

Output to

  • Plot to file  - This option sends the output to a file. this file can then be sent to a printer or be edited in a drawing program.
  • Preview mode  - In preview mode, the output is displayed in a separate preview window. If the output consists of more than one page, the pages are displayed one after the other in the preview window. Click in the window with the mouse, or press any key to get the next page. after the last page has been displayed, the window disappears. Press 'q' to abort the display of a series of pages.

    Paper Width, Paper Height

Plot 1D

Options specific for 1D plots.

 Axis options

     Draw axis

  • None
  • X - Draw X axis

    • Parameters X axis

    Units

    For 1D plots, the units are are only used for the x-axis.
  • PPM
  • Herz
  • Seconds
  • Channels

Spectral Plotting Limits

    The region of the spectrum that will be plotted. Left and Right are given in ppm or channels (depending on the axis units), and define the horizontal size. Bottom and Top define the vertical size. If the user does not modify these numbers, their values correspond to the current zoom settings.

Spectrum Scaling


     

     Scaling

  • Auto  scale - Automatically scale plot on paper. Use the whole plotting region.
  • Fixed Hz/cm - Scale according to a fixed hz/cm. If needed, the plot is split into more than one page.
  • Fill paper - Fill the whole page with the plot (for 1D, this is the same as Auto scale)

    • X Hz/cm

     This value is used if Fixed Hz/cm scaling is selected.

Peak Picking

    Peak Pick

  • No - disable peak picking
  • Yes - enable peak picking

    • Sign

  • '-' Pick negative peaks only
  • '+/-' Pick all peaks
  • '+' Pick positive levels only

    • Threshold

    The peak picking threshold.  This is the level that decides which peaks will be selected for picking. Below this level, nothing will be picked. For negative peaks, the negative value of the threshold is used.

X axis parameters

Autoscale tick mark

    With this option ON, the number of tick marks, labels, and also the number of grid lines for 2D files, is calculated automatically.

Tick marks

    The X axis has three kinds of tick marks, short, medium and long. If autoscale is Off,  these options define how many of these marks / unit (=ppm, Hz, etc) are drawn, and what sizes should be used.

Labels/ppm

    The number of (tick mark) labels per unit, if autoscale is Off.

Y offset axis (cm)

    The distance of the X axis to the spectrum in cm.

Y offset labels (cm)

    The distance of the bottom of the labels to the spectrum in cm.

X offset text (cm)

    The offset of axis text relative to the left side of the spectrum.

Y offset text (cm)

    The offset of axis text relative to the bottom of the spectrum.

Text for x axis.

    The text that will be drawn below the x axis.

Spectral Plotting Limits

    Left, Right, Bottom, Top

    The region of the spectrum that will be plotted. Left, Right, Bottom and Top are given in ppm or channels, depending on the axis units. If the user does not modify these numbers, their values correspond to the current zoom settings.

Spectrum Scaling


     

    Scaling

  • Auto - Make the plot as big as possible, to fit on one page. Aspect/ratio is kept.
  • Hz/cm - Plot with fixed Hz per cm. If needed, the plot is split into more than one page.
  • Fill - Fill all available space with the plot.

    X Hz/cm, Y Hz/cm

    If scaling is set to Hz/cm, these values define how many Hz will be plotted for each cm.

Grid Options

    Grid can be no, yes and auto. I Grid is no, no grid will be drawn. If grid is yes, define how to draw a grid.  Xline/unit and Yline/unit define how many grid lines per current unit (= ppm,Hz,etc) are drawn. For example, if Xline/unit is 2, and the unit is ppm, then a grid line is drawn every 0.5 ppm. If grid is set to auto, the number of lines/unit will be computed automatically.

    Xline/unit, Yline/unit

    Define the number of grid lines that will be drawn for each ppm in the X and in the Y direction, respectively. The positioning of this lines is automatic and will correspond as much as possible to the tick marks.
