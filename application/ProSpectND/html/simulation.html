Simulation

Getting started

    The simulation module is based on the popular Laocoon program for the analysis of isotropic NMR spectra for spin 1/2 systems (L. Cassidei and O. Sciacovelli). To get it started, use Options->Simulation. First, one is asked for a block number, to display the simulated spectrum. If the simulated spectrum is to be compared with a reference spectrum (which is assumed to be present in block 1) chose block 2. If block 1 is not in use and a spectrum is simulated without a reference spectrum, chose block 1.

    Next, the simulation popup menu is displayed. Keep this popup menu present during the whole simulation, otherwise parameters will get lost. If a previous run has been saved, parameters and settings can be restored from disk with Read Parameters.  The program can display the combined result of 12 independent spin systems. Each spin system can contain up to 10 spin groups.  A spin group contains one or more equivalent nuclei. Spin system can only be manipulated one at a time.


     

General Parameters

    If no experimental spectrum is present, use General Parameters to modify the characteristics of the simulated spectrum. The parameters reflect the settings of the spectrum in block 1. Make sure that this spectrum is properly calibrated, before starting the simulation.


     

       

Units

    Use PPM or Hertz for the chemical shifts. PPM is only valid to define shifts and for some output options. All the work is done in Hertz.

Spin Systems

Number of spin systems

In many case, only one spin system will be simulated (the default). If more spin systems are required, set the total number here.

Select

Select the active spin system. That is the spin system that can be manipulated.

Visisble

Select which spin systems are visible.

Scale

Set a scalings factor for each spin system. Normally, these factors are all 1.000. For a simulation of a mixture, different scaling factors can be applied. For example: if the spectrum of a reducing monosaccharide is simulated. The simullation of the alpha component is loaded in spin system 1, and the beta component in spin system 2. The scaling factors for 1 and 2 can be set to the apha/beta ratio.


 

Set up Spin System

Number of Spin Groups

    With Number of Spin Groups, the number of spins (or spin groups) is selected. This number ranges from 2 to 10. A Spin Group is either single spin or a set of equivalent spins, like the protons in CH3.

Spins and Couplings

    Now with Spins and Couplings, add the proper values. In our example we have two spins, and one coupling. If there are more spins, all possible coupling can be edited, although some obviously should be kept 0. Note that shifts are added in Hertz, not PPM, if Units is set to Hertz. Select an item and press Shifts and Couplings to change its value.


     

    Use Set Number of Spins for Shift to set the number of spins in a spin group, for example 3 for CH3. Use Set Isotope Number for Spin(s) to set the isotope identifier. For example, use 1 for protons and 2 for Carbon. The value of the isotope identifiers do not matter as long as different isotopes have unique identifiers.

    Finally press the Plot button to calculate and display the spectrum. Remember to use Plot whenever a change to the simulated spectrum has taken place and the spectrum should be refreshed.
     

Set up Iteration

Transitions

    Select a peak in the simulated spectrum and press the left mouse button. A popup window shows the selected transition(s) and the corresponding frequency. This frequency can be changed to a value that corresponds with a peak in the recorded spectrum. Check the frequencies that should be updated. Repeat this for all transitions that should be shifted to a different position.

    New transitions can be added, and transitions can be changed with the Transitions option. Also use this option to remove a transition or to clear all transitions.

Iteration Parameters

    It is often not a good idea to iterate all parameters at the same time. With Iteration Parameters, one can select the shifts and couplings that should be iterated. In our example, both shifts and their couplings in a two spin system are marked for iteration.

Iterate

    Usually, the simulated spectrum will not completely match the recorded one. To improve the match, one can use iteration. The first thing to do is to assign transitions to a frequency. This can be done interactively with the mouse, or with a popup menu. The next thing is to select chemical shifts and couplings that can be iterated. Its is very important to select only those shifts and couplings, that are dependent on the selected transitions. Otherwise, the iteration algorithm will change these values randomly. Use show dependencies, to check if the iteration parameters are chosen well.

Least squares fit

    Use least squares to fit calculated and defined transitions. Works fast if the iteration parameters are relatively independent (see also parameter dependencies). If both shifts and couplings are fitted, output can be terrible. In that case use different algorithm.


     

Simplex routine

    Use simplex routine for spectral refinement. Slower than least squares fit, but uses full eigenvalue calculations for the fit. Both shifts and couplings can be fitted concurrently.

    Trace:

    Set the amount of output. NO = show no results, Lowest = show if result is better, All = show all results.
     

    Re-assign transitions:

    When two transitions swap positions, their numbers change. If this option is set to YES, this change is included in the current iteration set.

Undo/Redo Iteration

    If the success of an iteration cycle is not obvious, use these buttons to toggle between the different spectra.
