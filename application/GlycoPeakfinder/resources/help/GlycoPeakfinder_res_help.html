Glyco-Peakfinder Help File
Annotate Peaks:
Calculation for all mass values from the peak list below is started using all parameters already set on the pages "mass", "residue", "ion/charge" and "modifications".
Reset all settings:
All settings manually performed on the pages "mass", "residue", "ion/charge" and "modifications" are cleared and default settings are set.
Reset form:
All settings manually performed on this form are cleared and default settings are set.
Help:
Starts Glyco-Peakfinder help files.
Residue list:
All residues that can be taken into account for composition analysis can be selected.
Increment mass:
Can be calculated from the mass of the residue by subtracting one water molecule.
Abbreviation:
Will be used for the representation of the calculation results.
Occurrence (min/max):
Each residue will be taken into account within the specified range given by min and max occurrence.
Calculation of A/X fragments:
For some residues an A/X fragmentation can be allowed. Therefore the checkbox has to be activated. Because of the huge number of these fragments, they can be selected separately for each residue (press button, will be implemented soon).
Other residues:
to allow residues other than the pre-setted up to three new residues can be specified. Additionally an increment mass and an abbreviation has to be given. Please keep in mind, that in case of persubstituted structures the mass of the modified residue(s) has to be entered.
