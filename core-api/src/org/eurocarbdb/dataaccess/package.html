The org.eurocarbdb.dataaccess package and its sub-packages provide the primary interfaces to data objects in the Eurocarb project. Many of the sub-packages of this package correspond to Eurocarb database schemas of (usually) the same name; the main ones are:

  • {@link org.eurocarbdb.dataaccess.core core} - the core database objects, including the 4 main core concepts of {@link org.eurocarbdb.dataaccess.core.GlycanSequence glycan sequence}, {@link org.eurocarbdb.dataaccess.core.BiologicalContext biological context}, {@link org.eurocarbdb.dataaccess.core.Evidence experimental evidence}, and {@link org.eurocarbdb.dataaccess.core.Reference reference}.
  • {@link org.eurocarbdb.dataaccess.ms ms} - data objects for mass spectrometry (MS)
  • {@link org.eurocarbdb.dataaccess.hplc hplc} - data objects for high performance liquid chromatography (HPLC)
  • {@link org.eurocarbdb.dataaccess.nmr nmr} - data objects for nuclear magnetic resonance (NMR)
